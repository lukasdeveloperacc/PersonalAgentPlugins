# PM Workshop Fixture: Backlog Grooming

## Input

The backlog has one large issue: "Improve onboarding". It includes copy changes, auth edge cases, profile setup, and first booking prompts.

## Expected Skill

`pm-plugin:backlog-groom`

## Expected Output Class

Propose-only GitHub update plan.

## Expected Proposals

- Split into Claude-executable issue candidates.
- Suggest labels and project status.
- Identify items that need brainstorm before TASK_SPEC.
- Mark irreversible close/deprioritization as human approval required.
