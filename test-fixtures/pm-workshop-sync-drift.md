# PM Workshop Fixture: Sync Drift

## Input

GitHub issue says In Progress, the TASK_SPEC says Draft, and a PR exists without a linked Claude handoff.

## Expected Skill

`pm-plugin:pm-sync`

## Expected Output Class

Drift report.

## Expected Findings

- Status mismatch between GitHub and Markdown.
- Missing PR to Claude handoff link.
- Required human decision before marking Done or closing.
